简单的用java做一个爬虫

发表于 2019-12-21 | 分类于 java | | 本文总阅读量 次

网络爬虫是什么

​ 网络爬虫（又称为网页蜘蛛，网络机器人，在FOAF社区中间，更经常的称为网页追逐者），是一种按照一定的规则，自动地抓取万维网万维网信息的程序或者脚本。另外一些不常使用的名字还有蚂蚁、自动索引、模拟程序或者蠕虫 。
​ 简而言之，网络爬虫就是帮助我们访问网络上的资源，我们一直以来都是用http协议访问互联网的网页，网络爬虫需要编写程序，我们这里同样用http协议访问网页。
​ 这里我们用java的HTTP协议客户端HttpClient这个技术，来实现抓取网页数据。

准备工作

jdk 1.8

IDEA

Maven

写一个入门程序

导包

第一步，我们需要从maven仓库中找到HttpClient这个包，pom文件如下添加

<dependency>
   <groupId>org.apache.httpcomponents</groupId>
   <artifactId>httpclient</artifactId>
   <version>4.5.2</version>
</dependency>

入门

然后我们先写一个入门小程序，比如让url指向www.taobao.com

public static void main(String[] args) throws Exception {
        //1.创建HttpClient对象
        CloseableHttpClient httpClient = HttpClients.createDefault();
        //2.输入网址，发起Get请求创建HttpGet对象
        HttpGet httpGet = new HttpGet("http://www.taobao.com");
        //3.按回车，发起请求并返回响应
        CloseableHttpResponse response = httpClient.execute(httpGet);
        //4.解析请求，如果状态码是200则输入响应实体
        if(response.getStatusLine().getStatusCode()==200){
            HttpEntity httpEntity = response.getEntity();
            String content = EntityUtils.toString(httpEntity,"utf8");

            System.out.println(content);
        }
    }

运行程序，观察控制台，打印出了完整的返回数据，很容易看出这是个html网页

实际上这就是我们模拟了http协议向淘宝发送了请求，返回给我们的一个完整的网页

请求传参

​ 我们发现，我们可以很轻易的使用httpclient来进行网页的访问，但是，我们写的入门代码有一定的局限性，比如下面这个例子，我们发现在虎牙搜索的时候，地址栏里会传入我们搜索的参数，我们怎么才能在访问的时候附带我们的参数呢

其实很简单，我们只要用到URIBuilder这个对象来帮我们

代码如下

public static void main(String[] args) throws URISyntaxException {
        //创建Httpclient对象
        CloseableHttpClient httpClient = HttpClients.createDefault();
        //创建URIBuilder
        URIBuilder uriBuilder = new URIBuilder("http://huya.com/search");
        //设置参数
        uriBuilder.setParameter("hsk","骚男");

        //创建HttpGet对象,传入带参URI
        HttpGet httpGet = new HttpGet(uriBuilder.build());

        //使用HttpClient对象发送请求，获取response对象
        CloseableHttpResponse response = null;
        try {
            response = httpClient.execute(httpGet);
            //解析响应
            if(response.getStatusLine().getStatusCode()==200){
                String content = EntityUtils.toString(response.getEntity(),"utf8");
                System.out.println(content);
            }
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            try {
                response.close();
            } catch (IOException e) {
                e.printStackTrace();
            }
            try {
                httpClient.close();
            } catch (IOException e) {
                e.printStackTrace();
            }
        }


    }

当我们的参数有多个的时候，我们也可以用多个uriBuilder.setParameter("键","值");来设置参数

改用Post请求

入门程序修改

这个时候有很多小伙伴会问了，怎么让客户端发送Post请求而不是Get呢

其实这个很简单，只要把入门代码中的HttpGet全部改成HttpPost就OK了

//创建HttpPost对象,传入带参URI
HttpPost httpPost = new HttpPost(uriBuilder.build());

带参数的Post请求

想让Post请求带参数，我们就得先声明一个List封装键值对

代码如下

public static void main(String[] args) throws URISyntaxException, UnsupportedEncodingException {
        //创建Httpclient对象
        CloseableHttpClient httpClient = HttpClients.createDefault();


        //创建HttpGet对象,传入URL
        HttpPost httpPost = new HttpPost("http://huya.com/search");

        //声明List集合，封装元素
        List<NameValuePair>  params = new ArrayList<NameValuePair>();
        params.add(new BasicNameValuePair("hsk","骚男"));

        //创建表单Entity对象
        UrlEncodedFormEntity formEntity = new UrlEncodedFormEntity(params,"utf8");

        //把Entity对象封装到HttpPost中
        httpPost.setEntity(formEntity); 

        //使用HttpClient对象发送请求，获取response对象
        CloseableHttpResponse response = null;
        try {
            response = httpClient.execute(httpPost);
            //解析响应
            if(response.getStatusLine().getStatusCode()==200){
                String content = EntityUtils.toString(response.getEntity(),"utf8");
                System.out.println(content);
            }
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            try {
                response.close();
            } catch (IOException e) {
                e.printStackTrace();
            }
            try {
                httpClient.close();
            } catch (IOException e) {
                e.printStackTrace();
            }
        }


    }

进一步

连接池

如果每次请求都要创建HttpClient，那么会造成频繁的创建和销毁的问题，那细心的你肯定想到了，连接池技术

就像java的jdbc连接池一样，利用连接池来减少创建销毁的次数来保证程序的效率更高，这里我们用到了PoolingHttpClientConnectionManager这个类

完整代码如下

package com.hwhjl.crawler.test;

import org.apache.http.HttpEntity;
import org.apache.http.client.methods.CloseableHttpResponse;
import org.apache.http.client.methods.HttpGet;
import org.apache.http.impl.client.CloseableHttpClient;
import org.apache.http.impl.client.HttpClients;
import org.apache.http.impl.conn.PoolingHttpClientConnectionManager;
import org.apache.http.util.EntityUtils;

import java.io.IOException;

public class HttpClientPoolTest {
    public static void main(String[] args) throws IOException {
        //创建连接池管理器
        PoolingHttpClientConnectionManager cm = new PoolingHttpClientConnectionManager();

        //使用连接池管理器发起请求
        doGet(cm);
    }

    private static void doGet(PoolingHttpClientConnectionManager cm) throws IOException {
        //从连接池获取httpClient对象
        CloseableHttpClient httpClient = HttpClients.custom().setConnectionManager(cm).build();
        //输入网址，发起Get请求创建HttpGet对象
        HttpGet httpGet = new HttpGet("http://www.taobao.com");
        //按回车，发起请求并返回响应
        CloseableHttpResponse response = httpClient.execute(httpGet);
        //解析请求，如果状态码是200则输入响应实体
        if(response.getStatusLine().getStatusCode()==200){
            HttpEntity httpEntity = response.getEntity();
            String content = EntityUtils.toString(httpEntity,"utf8");

            System.out.println(content);
        }
        response.close();
    }
}

小结

相信通过上述代码的编写，对java写一个小爬虫就有了初步的认识

其实用java写爬虫咱么可以借助很多第三方包，比如最常见的jsoup

对于爬虫的进一步学习，可以从jsoup开始
